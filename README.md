# ☕ Library Management System - Java

## 📌 About the Project

This project is a Java application developed to manage a library system integrated with a relational database.

The system connects to the database and performs operations such as:

- Registering clients
- Registering books
- Managing loans
- Recording returns

The application communicates with the database using JDBC.

---

## 🏗 Project Structure

The project follows a structured approach separating responsibilities such as:

- Database connection
- Data manipulation
- Business logic

This improves organization and maintainability.

---

## 🔌 Database Integration

The system connects to the previously created library database.

It performs:

- INSERT operations
- SELECT queries
- Updates
- Loan and return registration

The connection is handled through JDBC.

---

## 🛠 Technologies Used

- Java
- JDBC
- SQL
- Relational Database

---

## ▶ How to Run

1. Configure your database.
2. Update database credentials in the connection class.
3. Compile the project.
4. Run the main class.
